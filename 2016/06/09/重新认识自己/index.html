重新认识自己

是不是很久没有跟自己对话了？你可以尝试回答一些问题开始，将你过去要会回避的问题写下来，问题如下。这些问题的答案你可以选择不和大家分享，但是我强烈地建议写下来，只有亲手写下来才能够更深刻认知自己，不能写出的想法终究是没想法。谨慎思考这些问题，不逃避不留空白，这是接受自己的第一步。

  • 请你介绍下你自己，你是一个什么样的人？

    大学生，脑洞大，敢尝试，乐意思考，有梦想，不愿随波浊流。
    爱生活，会运动，敢拼搏，敢说敢做，有爱心，知道人情世故。

  • 你有什么理想吗？这个理想是怎么形成的？
    成为一名Geek，能够处理好复杂代码和懂设计工作（偏工业设计和模型设计类）
    要想达到目标，因此我需要：

  1. 学习计算机语言（PHP JAVA C++等）和熟知计算思维以及数据结构，在当前大数据云平台下处理好数据的关系，深挖后端工程。
  2. 要想让第一个目标更加容易和轻松，我百分之两百的需要学习好一门外语Eg：English，if can get a CET6 is better。（一定要坚持学习英语，切勿因为过CET而丢下，其中的口语！口语！口语尤为重要）
  3. 学好机械设计专业的课程，深挖实践课程，达到能够轻松做一个毕业设计，同时自己要成为答辩人。熟练掌握一款三位软件，尝试画出一个模型的整体。（从大到小）
  4. 青年更要活得开心，不悲观。多交良师益友、陶冶情操、可文艺可逗比更要学会装逼。
  5. 要养成善于记笔记的习惯，尝试多谢总结分析分享经验，多看书（互联网、管理）。
  6. 不要放弃不要因困难就退缩，在成功前一秒再坚持一秒。（未来已经来领，只是尚未流行）

  • 你理想的伴侣关系是什么样的？你自己的在这个伴侣关系中扮演什么样的角色？要承担什么样的责任？
    这个问题的答案需要用一生来回答。伴侣关系可以是简单的，长相思守的，质朴的；我扮演一个丈夫和父亲的角色，对妻有原则、对孩有要求；要去承担一个男人的责任和担当。

  • 你理想的事业是什么，你正在做的工作符合你的事业理想吗？这份工作对你的意义是什么？
    学好基本技能，以后少求人。多靠本领吃饭。现在是学生，会的太少，不会的太多，知道的太少，不懂得太多。正如交大张杰校长说：“上大学的过程是从不知道到知道再到不知道的过程。一开始是不知道自己不知道、逐渐才知道自己不知道；后来又不知道自己知道、再后来才能知道自己知道；到最后是以为自己都知道，才知道自己还有更多不知道。”

  • 你对亲子关系怎么看？对你来说，什么是一个理想的父亲（母亲），你期望自己成为这样一个理想父亲（母亲）吗？
    理想的父亲：
    现在作为一个理想主义者，在物质上，当然是给孩子和妻子创造更好的物质条件，让他们不需要太为生活而生活去操心；在精神上要有激情和热血。有一个幸福美满和谐欢乐的家庭。

  • 你对钱怎么看，你认为赚到多少钱是足够的？如果你明天一早醒来，已经有足够的钱，你将会如何继续安排自己的生活？
    初入职场，最重要的事就是提升自己的能力，而不是去做自己喜欢的事情，这个阶段钱能够养活自己就行。等你到了技能等身、拔剑四顾的阶段，尽快把这些技能转化成银元，这个阶段，不要对自己说，钱我看的很淡。照顾好自己，照顾好家人，为他们提供更好的生活条件才是你应该做的事情。等财富稍微自由一些，会突然发现，世界变得更加广阔，很多顾虑、担心、畏缩消失不见了，这时候，就到了为理想编程的时候！（为了他早日实现而Fighting！！）

  • 对你来说，什么是理想的性生活？什么是理想的性道德，在你的性道德观中，什么样的性生活是禁忌的，需要避免的，什么样的性生活是美好的，需要得到鼓励和发展的？
    这个问题略过好嘛，两个人知道就好。/害羞

  • 你的择友标准是什么？什么样的人你会愿意，什么样的人你会拒绝和他交往？

  1. 具有相同的价值观
  2. 有想法，愿意付出实践
  3. 低调做人，高调做事
  4. 愿意学习新知识和敢于新学一门自己必要的技术
    …… 这个问题涉及到很对因素，我会另写一篇。
  • 你对死亡怎么看？你希望自己活到多少岁，你准备怎么度过从现在到死亡的这段时间？如果你要立遗嘱，这份遗嘱怎么写？
    这个问题想开就好，不必要太纠结。生前需要做好做完自己应该做的部分。唯有不虚度年华不浪费光阴才可对得起出自农村的我。无需遗嘱

以上的这九个问题摘自《很少人能顺畅回答这9个问题——心理治疗刚开始医生常常会先问你的 》by 李孟潮。
